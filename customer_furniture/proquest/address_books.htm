Address Books
 
You can move between MY address book and Global address book by using the links on the right side of the Address book tab.
 
The Global address book
 
Global list – contains all ProQuest office contacts.
  
To view click “View Global Address Book” and then “Show all Global Addresses”.
 
When using this global book the system will only enter an address, you must specify a contact at the selected destination.
 
Only Super Users can add addresses to the Global book, all requests should be sent to Jo Uttley.
 
My address book
 
All individual contacts i.e. distributor addresses.
 
To view all addresses click “Show my full address list”.
 
To add a new address to your personal address book click “new”.
 
Fill all of the address details into the form, along with a contact telephone number for the recipient and “add” to my address book.
