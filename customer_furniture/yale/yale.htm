From 10th June 2010 we switched the Yale ‘marketing webshop’ service to a new online ordering system, from which literature and promotional goods can be ordered.   

You have been redirected to this system, which is accessible through...

   www.sprintexpress.co.uk/yale

or alternatively through the link in...

    www.yaledealer.com.

Users will have to register to gain access to the system by clicking on the link provided on the right and entering the Dealerships Yale Dealer Code. 

The main benefits of the new system will include the ability for users to:

  • See the revision number of literature items. This will enable users to check if new updated literature needs to be ordered
  • Track and Trace orders placed through the system
  • Add regular destinations to a personal address book to save time when placing orders
  • View Freight Tariff Lists and receive more accurate estimated freight charges when placing orders

User guidelines will be available to download from the homepage once logged in.
