HOW TO USE THE TSE TICKET MOBILE APP

The TSE ticket mobile app allows TSEs to submit support requests on behalf of Agents. You can find it at URL...

  http://tinyurl.com/wuticket

To use the app...
  1. Go to the URL above.

  2. Enter the 4-character Terminal ID of the Agent for whom you want to submit the support message, then press the Send button (depending on your browser, you may be able to press the Enter key instead).

  3. If the Terminal ID is recognised, the next screen displays the Agent Name. Check that you have selected the correct Agent.

  4. Click Select Topic to display the available titles for the conversation. Choose a title. If you choose 'Other', enter your title in the text box that appears.

  5. In the TSE User ID text box enter your TSE User ID followed by a space followed by at least the first 2 characters of your password (case sensitive).

    To have your device remember your TSE User ID for next time, move the Remember? toggle button to Yes. If you did that previously your TSE ID is displayed and you only need to enter the first 2 character of your password.

  6. In the Message text box, enter the text of the message you are sending on behalf of the Agent.

  7. Press the Send Message button. Your message will be sent to the Controller.

NOTE: When the controller closes your conversation you will receive a notification by email. Check the Controller's response and if appropriate inform your Agent that the query has been resolved.
